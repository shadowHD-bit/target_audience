Локальный обменник под брендом
ICP-1

Клиенту нужен быстрый запуск локального обменника с языком, доменом, поддержкой и понятной зоной ответственности

Формат
WL SaaS / self-host
Функции
Котировка, QR, история, статусы, 14 сетей
Роли
Основатель, операции, CTO, комплаенс
Обменные платформы
Расширение exchange-контура
ICP-2

Клиент уже работает с обменами и хочет расширить активы, курсы, поставщиков, виджет, API или новый frontend

Формат
WL / API / self-host
Функции
Курсы, лимиты, статусы, история, отчеты
Роли
Основатель, продукт, CTO, поддержка
Крипто-трафик и медиа
Монетизация аудитории
ICP-3

Клиент хочет превратить аудиторию в обменные операции под своим брендом, а не отправлять ее на чужой сайт

Формат
WL SaaS / домен клиента
Функции
Комиссия, отчеты, GA4, SEO, страницы пар
Роли
Основатель, рост, SEO, партнерства
Кошельки и портфели
Встроенный обмен

Формат

Функции
Deep links, статусы, комиссии, отчеты
Роли
Продукт, CTO, партнерства
